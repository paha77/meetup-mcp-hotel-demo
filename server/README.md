# MCP server - Software Craft and Testing Community Meetup @ Vivid Planet

This is an MCP server client that simulates the backend of a hotel.

## Setup

1. Install dependencies:

    ```bash
    npm install
    ```

2. Set up the database with dummy data:

    ```bash
    npm run fixtures
    ```

3. Start the development server:

    ```bash
    npm run dev
    ```

4. The MCP server runs at  `http://localhost:3000/mcp`
